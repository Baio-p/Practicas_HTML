COMPORTAMIENTO DE ETIQUETAS

  • Etiqueta de enlaces, links o anclas < a >

    • la primer etiquta que usaremos sera para
    para poner un link en una pagina web.
    Al abrir la etiqueta usaremos el atributo HREF="",
    escribiendo entre las comillas el enlace que queremos insertar en nuestra pagina, cerramos el inicio de la
    etiqueta y despues colocaremos una descripcion para el link antes de cerrar nuestra etiqueta.

    soy un enlace a CERN

  • etiqueta < abbr >

    • Se usa para asignar una abreviatura en la pagina. Se abre la etiqueta con el atributo TITLE=""
    escribiendo entre las comillas la descripcion para la abreviatura que queramos colocar; cerramos
    el inicio de la etiqueta y despues colocamos la abreviatura para despues cerrar la etiqueta.
