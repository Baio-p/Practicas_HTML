COMPORTAMIENTO DE ETIQUETAS

  • Etiqueta < a >

    • Se usa para anclar un url, tambien conocido como "LINK", en nuestra pagina.
    Abrimos la etiqueta con el atributo HREF=" ", escribiendo el url dentro de las comillas,
    despues escribimos alguna descripcion sobre el link anotado para que aparezca de esta forma y
    ahora podemos cerrar la etiqueta.
    Ejemplo:

    soy un enlace a CERN

  • etiqueta < abbr >

    • Se usa para asignar una abreviatura en la pagina con una descripcion flotante al colocar el cursor
    sobre ella. Abrimos la etiqueta con el atributo TITLE=" ", escribiendo entre las comillas la descripcion
    para la abreviatura que queramos colocar, despues escribimos la forma en que queramos poner la abreviatura
    y asi poder cerrar nuestra etiqueta.
    Ejemplo:

    html

  • etiqueta < adress >

    • Define la información de contacto del autor/propietario de un documento o artículo.
    Ejemplo:

    escrito por Benja

  • etiqueta < article >

    • Un artículo debe tener sentido por sí mismo y debe ser posible distribuirlo independientemente del resto
    del sitio.
    Fuentes potenciales para el elemento article:

    • Publicación en el foro
    • Entrada en el blog
    • Noticia

    NOTA: < article > no se muestra como algo especial en un navegador.
    Sin embargo, puede usar CSS para darle diseño
    Ejemplo:

    ejemplo de article

  • etiqueta < aside >

    • Define algún contenido además del contenido en el que se coloca. El contenido aparte debe estar
    indirectamente relacionado con el contenido circulante.
    Sugerencia: < aside > se coloca a menudo como una barra lateral en un documento.
    Nota:Este elemento no se muestra como algo especial en el navegador. Sin embargo, puede usar CSS para
    diseñar el aside.

  • etiqueta < form >

    • Es un contenedor para diferentes tipo de elementos de entrada, como: campos de texto, casillas de
    verificación, botones de opción, botonees de envío, etc.

    • <input type="text">

      • Mustra un campo de entrada de texto de una solo linea

    • <input type="radio">
      • 1 2 3

      Muestra un botón de radio(para selecionar una de muchas opciones)

    • <input type="checkbox">

      • Muestra una casilla de verificación (para selecionar cero o más opciónes)

    • <input type="submit">

      • Muestra un botón de envio(para enviar el formulario)

    • <input type="button">

      • Muestra un botón en el que se puede hacer clic

  • etiqueta < header >

    • Representa un contenedor de contenido introductorio o un conjunto de enlaces de navegación. Normalmente contiene:

    • Uno a más elementos de encabezado (< h1 > a < h6 >)
    • Logotipo o icono
    • información de autoría

      • NOTA: Puede tener varios < header > en un documento HTML. Sin embargo,
      no se pueden colocar dentro de un < footer >, < adress > u otro < header >
      ejemplo:

    HOLA MUNDO

    Por Benja

    por ejemplo, un titulo de sección y un subtitulo adjunto.
    ejemplo:

    EL COMIENZO...

  • etiqueta < img >

    • Se utiliza para incrustar una imagen en una página HTML.
    Técnicamente, las imágenes no se insertan en una página web; las imágenes están vinculadas a páginas
    web. La etiqueta < ima >crea un espacio de espera para la imagen de referencia.
    Tiene atributos obligatorios:

    • src: especifica la ruta a la imagen
    • alt: especifica un texto alternativo para la imagen, si la imagen por algún motivo no se puede mostrar

  • etiqueta < map >

    • Se utiliza para definir un mapa de imagen, es una imagen con áreas en las que se puede hacer clic. El atributo de nombre requerido del < map > esta asocioado con el atributo usemap de < img > y crea una relación entre la imagen y el mapa.
    ejemplo:

    soy el mapa

  • etiqueta < nav >

    • Define un conjunto de enlaces de navegación. Toma en cuenta que NO todos los enlaces de un documento deben estar dentro de un < nav >. Este elemento está destinado solo para el bloque principal de enlaces de navegación.
    ejemplo:

    youtube google

  • etiqueta < p >

    • Define un párrafo.
    Los navegadores agregan automáticamente una línea en blanco antes y después de cada < p >

    ejemplo de < p >

  • etiqueta < scrypt >

    • Se utiliza para incrustar un script del lado del cliente JavaScript.
    Contiene declaraciones de secuencias de comandos o apunta a un archivo de secuencia de comandos externos a través del
    atributo src Los usos comunes de JavaScript son manipulación de imágenes, la validación de formularios y los
    cambios dinámicos de contenido.
    Nota: Cuando un navegador lee una hoja de estilo, formateará el documento HTML de acuerdo con la
    información de la hoja de estilo. Si se han definido algunas propiedades para el mismo selector
    (elemento) en diferentes hojas de estilo, se utilizará el valor de la última hoja de estilo leída
    (consulte el ejemplo a continuación).

    ejemplo de < scrypt >

  • etiqueta < style >

    • Se utiliza para definir la información de estilo CSS de un documento
    Dentro de < style >, especifica cómo deben mostrarse los elementos HTML en un navegador.

  • tipos de diferenciado

    • Para poner estilos a ciertas etiquetas en CSS se pueden aplicar por:

    • class: (."nombre") Tu puedes darle un nombre a una o varias etiquetas por algun nombre
    • tag: ("nombre de etiqueta") Son las palabras reservadas a las etiquetas y actua sobre toda las etiquetas con ese nombre
    • id: (#"nombre") Son nombres para comandos (como los comandos en scrypts)
