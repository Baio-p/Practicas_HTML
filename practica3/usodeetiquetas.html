    • etiqueta < img >

      • Se utiliza para incrustar una imagen en una página HTML. Técnicamente, las imágenes no se insertan en una página web; las imágenes están vinculadas a páginas web. La etiqueta < ima >crea un espacio de espera para la imagen de referencia. Tiene atributos obligatorios:

      • src: especifica la ruta a la imagen
      • alt: especifica un texto alternativo para la imagen, si la imagen por algún motivo no se puede mostrar
